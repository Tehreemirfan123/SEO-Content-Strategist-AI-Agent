# SEO Content Strategist AI Agent

An intelligent multi-agent AI automation system that transforms a single Google Form submission into a complete SEO content strategy, SEO-optimized blog article, social media content, and quality assurance report using Make.com and Large Language Models (LLMs).

---

## Project Overview

**SEO Content Strategist AI Agent** is an end-to-end AI workflow automation system designed to eliminate the repetitive and time-consuming process of SEO content planning.

Instead of manually researching keywords, creating outlines, writing metadata, generating blog articles, and preparing social media posts, users simply submit a Google Form. The workflow orchestrates multiple specialized AI agents that collaborate to produce a complete, publication-ready SEO content package.

This project demonstrates practical implementation of **Agentic AI**, **workflow automation**, and **business process orchestration** using Make.com.

# Multi-Agent Architecture

| Agent                      | Responsibility                                         |
| -------------------------- | ------------------------------------------------------ |
| SEO Strategy Agent         | Search intent analysis, keyword planning, SEO strategy |
| Title Generator Agent      | Generate high CTR SEO titles                           |
| Outline Generator Agent    | Create H1–H3 article outline                           |
| Content Writer Agent       | Write complete SEO article                             |
| Metadata Generator Agent   | Generate SEO metadata                                  |
| FAQ Generator Agent        | Generate FAQs and internal links                       |
| Content Repurposing Agent  | Create social media posts                              |
| SEO Quality Review Agent   | Review and score generated content                     |
| Content Optimization Agent | Improve content based on review                        |
| Decision Engine            | Control optimization workflow                          |
